# ADR 002 — tRPC v11 with createTRPCContext

**Date:** 2026-04-01  
**Status:** Accepted

## Context

tRPC v11 changes the React integration API. The old `createTRPCReact()` hook-factory pattern is deprecated. v11 uses `createTRPCContext` and integrates directly with TanStack Query v5.

## Decision

Use `@trpc/tanstack-react-query` v11 with `createTRPCContext`. The provider wraps `QueryClientProvider` and the tRPC client is initialized with `httpBatchLink`.

## Consequences

- `trpc.useQuery` / `trpc.useMutation` are available throughout client components
- Server components call procedures directly (no HTTP overhead)
- Batching is on by default — multiple queries in one render coalesce into one HTTP request
- Context is typed end-to-end; no `any` casts needed
